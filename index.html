01. Prérequis & Installation de Base

Avant d'intégrer des fonctionnalités avancées d'IA ou des scripts complexes, votre jeu doit disposer d'une base parfaitement propre et à jour.

⚠️
Important : Faites toujours une copie de sauvegarde de votre dossier de jeu original avant d'installer le moindre mod.

Les éléments indispensables à télécharger

📁 LSPDFR Initial Setup

Téléchargez la version "Manual Install" de préférence pour garder un contrôle total sur vos fichiers.

⚙️ Rage Plugin Hook

Généralement inclus dans LSPDFR, mais assurez-vous d'avoir la toute dernière version disponible sur leur site.

💡
N'oubliez pas d'installer également RageNativeUI (le fichier .dll va à la racine de votre jeu) pour permettre l'affichage des menus en jeu.

02. Plugins Recommandés & Dépendances

Pour faire fonctionner l'écosystème d'immersion et les interactions avec les suspects de manière fluide, certains frameworks tiers sont indispensables.

Vérification des frameworks requis

Certains mods avancés s'appuient sur des outils de tracking de dégâts ou de gestion de scripts pour l'IA comportementale. Assurez-vous d'avoir correctement placé DamageTrackingFramework.dll.

Configuration des dépendances logicielles

Dans vos fichiers de configuration d'initialisation de scripts (comme pour l'autoloader ou les gestionnaires de plugins), veillez à ce que la détection des dépendances cible le répertoire racine des plugins standard.

UltimateBackup.ini / CustomRestriction.txt 
// Vérification de la présence des frameworks requis à l'initialisation
Need 'plugins\DamageTrackingFramework.dll' 'DamageTrackingFramework' '4'
🚨
Erreur courante : Ne confondez pas le dossier plugins\ général avec le sous-dossier plugins\LSPDFR\ pour ce framework spécifique. Si le fichier est mal placé ou mal nommé (par exemple en l'orthographiant "DamageTracker" au lieu de "DamageTracking"), vos dépendances ne chargeront pas et le jeu plantera au démarrage de RagePluginHook.

03. Configuration de l'IA Behavior

Ajustement des comportements des patrouilles et des réactions des suspects lors des poursuites ou des interpellations.

Ouvrez le fichier de configuration de l'IA comportementale et ajustez les variables suivantes pour obtenir des réactions réalistes, évitant que les suspects ne tirent instantanément sans sommation.

lspdfr/PoliceScanner.ini 
[AmbientCopAI]
AggressivenessMultiplier = 0.7
ResponseTimeSeconds = 2.5
UseSmartCoverBehavior = true

[SuspectAI]
SurrenderChanceOnTaser = 0.65
FleeOnFootProbability = 0.40

04. Résolution des Problèmes (Crashs)

Si votre jeu se fige ou retourne brusquement sur le bureau lors du chargement, vérifiez les points suivants :

  • Augmentez le Plugin Timeout Threshold dans les options de Rage Plugin Hook à 10000 (10 secondes au lieu de 1000).
  • Vérifiez que vous possédez bien la version correspondante de DamageTrackingFramework.dll dans votre répertoire principal plugins\.
  • Consultez le fichier RagePluginHook.log situé à la racine pour identifier précisément quelle ligne ou quel plugin a provoqué l'interruption.
